# react-slider

## Getting started

To get the frontend running locally:
- Clone this repository. You will need node and npm installed globally on your machine.

## Install

    $ git clone https://github.com/manvendra22/react-slider.git
    $ cd react-slider
    $ npm install

## Start & watch

    $ npm start

## Build

    $ npm run build

## Start storybook

    $ npm run storybook
